@BeforeEach
- Se ejecuta antes de cada test para preparar el entorno (abrir navegador, ir a la web, etc.).

assertTrue
- Comprueba que una condición es verdadera.

findElement() vs findElements()
- findElement() → devuelve un elemento y falla si no existe.
- findElements() → devuelve una lista y NO falla (lista vacía si no hay nada).

¿Por qué usar LoginPage?
- código más limpio, reutilizable y fácil de mantener.
